# Components Used

## Hardware Components

* Arduino UNO
* MQ-3 Alcohol Sensor
* Eye Blink Sensor
* Relay Module
* Buzzer
* Breadboard
* Jumper Wires
* 12V Power Supply
* Gear motor

## Purpose of Components

### MQ-3 Alcohol Sensor

Detects alcohol concentration from the driver's breath.

### Eye Blink Sensor

Monitors eye closure and blinking activity to detect drowsiness.

### Relay Module

Controls vehicle ignition based on sensor readings.

### Buzzer

Provides audible alerts when unsafe conditions are detected.

### Arduino UNO

Processes sensor data and controls system responses.
