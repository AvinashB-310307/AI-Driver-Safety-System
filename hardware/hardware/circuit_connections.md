# Circuit Connections

## MQ-3 Alcohol Sensor

| MQ-3 Pin | Arduino |
| -------- | ------- |
| VCC      | 5V      |
| GND      | GND     |
| A0       | A0      |

## Eye Blink Sensor

| Sensor Pin | Arduino |
| ---------- | ------- |
| VCC        | 5V      |
| GND        | GND     |
| OUT        | D2      |

## Buzzer

| Buzzer Pin | Arduino |
| ---------- | ------- |
| Positive   | D8      |
| Negative   | GND     |

## Relay Module

| Relay Pin | Arduino |
| --------- | ------- |
| VCC       | 5V      |
| GND       | GND     |
| IN        | D7      |
