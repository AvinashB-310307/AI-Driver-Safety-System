# Smart Driver Safety System

## Introduction

Road accidents caused by drunk driving and driver drowsiness are major safety concerns. This project introduces a smart safety system that detects alcohol consumption and drowsiness using sensors and alerts the driver before accidents occur.

## Objectives

- Detect alcohol consumption
- Detect driver drowsiness
- Alert the driver using a buzzer
- Prevent unsafe vehicle operation

## Components Used

- Arduino UNO
- MQ-3 Alcohol Sensor
- Eye Blink Sensor
- Relay Module
- Buzzer

## Working Principle

1. Alcohol sensor checks alcohol level.
2. If alcohol exceeds threshold:
   - Buzzer activates
   - Ignition is blocked
3. Eye blink sensor monitors eye activity.
4. If prolonged eye closure is detected:
   - Buzzer activates
5. Driver receives warning before unsafe driving conditions occur.
